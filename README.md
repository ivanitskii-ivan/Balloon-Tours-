# 🎈 Balloon Tours

Адаптивный лендинг для компании, организующей туры на воздушных шарах. Пет-проект, написанный без JS-фреймворков — только HTML, SASS и CSS.

## 🔗 Демо
[Открыть сайт](#) ← вставь свою ссылку

## Стек
HTML5 · SASS · CSS3

## Что реализовано
- 3D flip-карточки туров (CSS perspective)
- Анимации при загрузке страницы
- Валидация формы на чистом CSS
- Видео-фон с наложением градиента
- Адаптивность от 320px до 1800px+
